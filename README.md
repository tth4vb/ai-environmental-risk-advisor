# AI Environmental Risk Advisor - Prototype

A community-focused environmental risk assessment tool for potential mining projects. This prototype demonstrates how communities can access independent, trustworthy information about mining projects to prepare for consultations and negotiations.

## Overview

This tool helps local communities and CSOs:
- Understand environmental risks from planned mining projects
- Translate complex technical documents into simple language
- Plan actions for each project phase
- Connect with verified technical advisors
- Generate documents for legal proceedings

## Features Demonstrated

### 1. Project Input Form
- Smart taxonomy for mining project details
- Pre-filled examples for common scenarios
- Focus on critical factors affecting communities

### 2. Risk Assessment Dashboard
- Traffic light system for environmental risks
- Data source attribution for transparency
- Actionable recommendations for communities

### 3. Technical Translator
- Converts complex reports to plain language
- Highlights key concerns and recommendations
- Sample water quality report translation

### 4. Action Planner
- Timeline of project phases
- Community rights at each stage
- Documents to request
- Actions to take

### 5. Data Visualizations
- Water risk mapping
- Community impact radius
- Project timeline
- Comparative analysis with similar projects

### 6. Expert Connections (Fake Door)
- Verified technical advisors
- Multi-language support
- Group consultation scheduling

### 7. Document Generator (Fake Door)
- Customized templates for community use
- Position statements
- Information requests
- Grievance submissions

## Running the Prototype

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the prototype.

## Design Principles

- **Trust First**: Independent data sources, transparency about limitations
- **Community Agency**: Tools that increase decision-making power
- **Accessibility**: Simple language, visual indicators, mobile-friendly
- **Evidence-Based**: Real data sources, documented methodologies
- **Action-Oriented**: Clear next steps, not just information
