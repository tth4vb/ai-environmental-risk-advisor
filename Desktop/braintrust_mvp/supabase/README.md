# Database Migrations

## Apply migrations manually

1. Go to Supabase Dashboard > SQL Editor
2. Copy contents of migration file
3. Run the SQL
4. Verify tables created in Table Editor

## Migrations

- `20251122000001_initial_schema.sql` - Initial database schema with users, documents, tags, and RLS policies
